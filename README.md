## 📌 Descripción

**Full Stack Ride App** es una aplicación móvil que permite a los usuarios solicitar viajes, gestionar pagos y visualizar rutas en tiempo real.

El sistema está diseñado con una arquitectura moderna, combinando frontend móvil con backend y servicios externos para ofrecer una experiencia completa de transporte digital.

---

## 🚀 Características

- 📱 Aplicación móvil multiplataforma
- 🚖 Solicitud de viajes en tiempo real
- 📍 Geolocalización de usuarios
- 💳 Pagos integrados con Stripe
- 🔐 Autenticación de usuarios
- ⚡ Interfaz rápida y responsiva
- 🗄️ Gestión de datos con PostgreSQL

## ⚙️ Instalación

1. Clonar repositorio:
```bash
git clone https://github.com/isairey/UberClon.git
cd UberClon
```
Instalar dependencias:
```
npm install
```
Iniciar proyecto:
```
npx expo start
```
